//////////A LIRE AVANT UTILISATION\\\\\\\\\\

Librairies utilisées:
libsdl1.2-dev
libsdl-gfx1.2-dev
libsdl-ttf2.0-dev

Avant utilisation, veuillez mettre les fichiers .asc des résolutions 75m et 5m dans le dossier data/data_75
et data/data_5. Pour la résolution 75, assurez vous que les fichiers .asc de la corse figure dans data_75.
Le fichier .asc de la résolution 1000m doit être dans le dossier du programme (fourni avec le code).

Exemple de compilation : gcc -o prog main.c partie1.c partie2.c propagation.c bouton.c -lSDL -lSDLmain -lSDL_ttf -lSDL_gfx -lpthread
